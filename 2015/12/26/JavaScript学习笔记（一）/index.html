JavaScript学习笔记(一)

JavaScript高级程序设计1-4章阅读笔记

JavaScript简介

javascript

  • 核心（ECMAScript）提供核心语言功能
  • 文档对象模型（DOM）提供访问和操作页面内容的方法和接口
  • 浏览器对象模型（BOM）提供与浏览器交互的方法和接口

JavaScript在HTML中使用

属性
  1. async:可选，立刻下载脚本，不妨碍页面中的其他操作（下载资源、等待加载其他脚本等）。
    • 只对外部脚本有效。
    • 如有两个async，则确保两者不相互依赖。
  2. defer：可选，表示脚本延迟到文档完全被解析和显示后执行。
    • 只对外部脚本有效。
    • 最好只包含一个延迟脚本。（不一定按照顺序执行，不一定先于DOMContentLoaded事件触发前执行）
    • 延迟脚本放在页面底部是最佳选择。
  3. src:可选，表示包含要执行代码的外部文件。
    • 如带有scr,则<script>``</script>标签里嵌入代码被忽略。
  4. type：可选，默认值为text/javascript。
  5. charset：可选。
  6. language
位置
  • <head>中：则必须等到全部Javascript下载、解析、执行后，遇到<body>标签才开始呈现内容。可能导致浏览器呈现页面出现延迟，页面出现前窗口一片空白。
  • </body>前：用户因浏览器窗口显示空白页面时间缩短而觉得页面打开速度加快。
  • <noscript>

JavaScript基本概念

  • 标识符：首字母小写，剩下每个单词首字母大写。如firstSecond、myCar。
  • 语句结尾加分号;。
  • 即使只有一条语句，也使用代码块。
  • 不推荐省略var的全局变量

数据类型

  typeof操作符检验基本数据类型。

  • 5种简单数据类型（也称为基本数据类型）：Undefined、Null、Boolean、Number和String。
  • 1种复杂数据类型：Object，Object本质上是由一组无序的名值对组成的。

    Undefined

      只有值undefined。undefined派生自null。
  • 未初始化/未声明的变量执行typeof都返回undefined
  • alert(null == undefined) //true

Null

  只有值null,表示空对象指针。

  • 执行typeof返回object

Boolean

  两个值ture,false。ture不一定等于1，false不一定等于0。

Number

  • 进制
    • 十进制
    • 八进制 0
    • 十六进制 0x(A~F可大写可小写)
  • 浮点数值
    • 0.1
    • 不要测试某个特定的浮点数值。如if(a + b = 0.3)×
  • 数值范围
    Number.MIN_VALUE，Number.MAX_VALUE。超出此范围，转换成(-)Infinity。访问这两个属性，可得到(-)Infinity。
  • NaN
    • 0/0返回NaN
    • 正负数/0返回(-)Infinity
    • NaN与任何值都不相等，包括NaN本身
    • isNaN()
      适用于对象，会先调用ValueOf()方法可返回 Boolean 对象的原始值。
  • 数值转换
    • Number():用于任何数据类性。
数字值 简单的传人传出
Booean值(false/true) 0/1
null 0
undefined NaN
字符串 前导零忽略(既不能转换二进制)
十六进制转换为十进制数
“”转换为0
其他转换为NaN

  • parseInt()：
    ①常用于处理整数。
    ②转换字符串时，忽略字符串前面的空格，直到找到第一个非空格字符。
    ③若第一个字符不是数字字符或负号，则返回NaN。
    ④不能转换为小数。
    ⑤ parseInt(“”,基数)：可严格准换为基数进制，因此任何情况下都使用基数。

  • parseFloat()：
    ①常用于处理小数。
    ②始终忽略前导零。即只解析十进制值，十六进制字符串转换为0。
    ③第一个小数点有效，第二个无效。

String

  • 字符字面量：特殊字符字面量如/n
  • 字符串特点：
    字符串一旦创建，值不可改变。要改变某个变量的保存的字符串首先要销毁原来的字符串，然后用一个包含新值得字符串填充该变量。
  • 转换字符串
    • toString():null/undefined值无此方法。
    • String()：null/undefined返回两个值字面量。

Object

每个实例都具有下列属性和方法

  • constructor
  • hasOwnProperty(propertyName):检验给定的属性在当前对象实例中是否存在。属性名以字符串形式指定。
  • isPrototypeOf(object):检查传入的对象是否是传入对象的原型
  • propertyIsEnumerable(propertyName)：检验属性是否可用for-in枚举。
  • toLocaleString()
  • toString()
  • valueOf()
理解参数

argument：函数体内可通过argument对象来访问参数数组，从而获取传递给函数的每一个参数。

typeof:操作变量或数值字面量。返回字符串。

  • "undefined"
  • "boolean"
  • "number"
  • "string"
  • "object"
  • "function"

Boolean():转型函数，返回一个Boolean值。
转换为false：空字符串；0和NaN；null；undefined。
if(NAME)自动执行Boolean转换。

ValueOf()方法可返回 Boolean 对象的原始值。

JavaScript变量、作用域和内存问题

两个类型的值

  基本类型值：简单的数据类型。Undefined、Null、Boolean、Number、String。存储于栈内存。
  引用类型值：可能由多个值构成。存储于堆内存中。Object。

复制变量值

  • 复制基本类型值：在复制的变量对象上创建一个新的值，然后把该值复制到新变量分配的空间中。
  • 复制引用类型值：将原变量对象的值复制到新变量分配的空间中，但这个值是个指针，复制操作结束后，两个变量将引用一个对象。（）复制的其实是指针， 因此两个变量最终都指向同一个对象。
    ※包含引用类型值的变量实际上是包含的一个指向该对象的指针。

传递参数

  ECMAscript中所有的函数的参数都是按值传递。

  • 传递基本类型值：被传递的值复制给一个局部变量(命名参数)
  • 传递引用类型值：将这个值在内存中的地址复制给一个局部变量，这个局部变量的变化会反映在函数的外部。

检测类型

  • typeof()：检测引用类型的值
  • instanceof()：检测引用类型是什么类型的对象

实行环境及作用

作用域链：内部环境可以通过作用域链访问外部环境。
延长作用域链：①try-catch语句的catch块 ②with语句
JavaScript没有块级作用域

  1. JavaScript的作用域完全是由函数来决定的，if、for语句中的花括号不是独立的作用域。
  2. 函数作用域:搜索顺序为①当前函数作用域 ②上层作用域 ③最后全局作用域 ④未声明

  3. 变量声明语句永远在该作用域里最先被执行。

  4. 函数作用域的嵌套关系是在定义时决定的，而不是在调用时决定的

JavaScript垃圾收集

  • 标记清除
  • 引用计数
    声明变量+赋给该变量引用类型值： 引用次数为1
    同一个值赋给另外一个变量： 引用次数+1
    包含这个值得变量取得了另外一个值： 引用次数-1
    引用次数变为0，侧无法再访问，释放其内存空间。
    但相互引用的对象的引用次数永远不会是0。
