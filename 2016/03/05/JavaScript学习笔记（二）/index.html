    返回原数组改变后的数组

  1. concat()
    基于当前数组所有项创建一个新数组

    基于当前数组中的一个或者多个创建当前数组

  3. splice()
    向数组中部插入项

    • 删除: splice(要删除的第一项的位置，删除的项数)
    • 插入/替换: splice(起始位置，删除的项数，n个插入的项)
位置方法

  两方法都接受2个参数：查找的项,(查找的起点位置)。找不到返回-1。

  1. indexOf()：从开头查找
  2. lastIndexOf()：从末尾查找
迭代方法

  每个方法都接受两个参数：每一项上运行的函数，(运行该函数的作用域对象——影响this值)。每个方法都先对每一项运行给定函数。
  函数接受三个参数：遍历的数组项的值，该项在数组中的位置，数组对象本身。

  • every()：每一项都返回true,则返回true。
  • some() ：任一项返回true，则返回true。
  • filter() ：返回该函数返回true的项组成的数组。
  • forEach()：没有返回值
  • map()：返回该函数结果组成的数组。

Function

  1. 函数是对象，函数名是指针。
  2. 没有函数重载（即同名函数，后者覆盖前者）
  3. 函数声明 ：代码开始执行之前，解析器就应经通过一个名为函数声明提升的过程，取读并将函数声明添加到执行环境中。
    函数表达式：函数位于初始化语句中。
  4. 函数作为值：函数的两个参数可以是函数，返回值可以是函数。
