Ife-2015-task2 任务笔记

任务1：输入词汇并分割并输出checkbox

HTML
  1. 输入框textarea,输入框内容提示placeholder
  2. 输入框后的按钮botton，输入完成
  3. 输入框下面的js验证文字
JS
  1. 事件监听
    • 输入框
      • focus:光标在输入框中
      • blur :js验证，读取输入框内容
    • 按钮
      • click：读取输入框内容并验证，无误则显示输入框内容成checkbox样式。

  2. 读取输入框内容方法 readHobbyArr():

    • 替换输入框内容的分隔符，将多项用空格分开 trim(text).replace([\s,，、;；]+/g, “ “)
    • 将多项储存为数组 text.split(“ “)

    • 数组中元素进行去重操作

    • 将生成的内容数组进行验证alterText(textArr)，并将结果显示在验证提示文字中

    • 返回textArr
  3. checkHobbyArr()
  4. showHobby()

任务2：倒计时

  1. 点击倒计时按钮后开始比较end.getTime()与start.getTime()的时间先后，确定是倒计时文字显示内容。
  2. 开始倒计时每隔1s执行一次倒计时文字更新window.setInterval(setCountdown, 1000);
  3. 将window.setInterval(setCountdown, 1000);赋给一个全局变量doCountdown，用于检查此时是否有循环函数正在执行。
  4. 倒计时为0或更新endTime后，清除正在循环执行的函数window.clearInterval(doCountdown)。
  5. 其它
    • getTime 方法的返回值一个数值，表示从1970年1月1日0时0分0秒（UTC，即协调世界时）距离该日期对象所代表时间的毫秒数。
    • Math.floor(x) 函数返回小于或等于数 “x” 的最大整数

任务3：轮播图

  1. 使用构造函数MovePic生成新的轮播图实例，初始可配置逆序/正序、是否循环、循环间隔时长。构造函数的原型具有默认配置MovePic.setDefaultData()。
  2. MovePic.renderCircle()根据图片数量加载对应的小圆点
  3. MovePic.move(this)因为页面初始定个在第一幅图上，所以在用setInterval使图片轮播时要先从整个页面的第二幅图开始。

HTML
JS
  1. 事件监听
    • 输入框内按下键盘
      • keybown:如果是按了上、下或回车键则对显示的光标进行相应操作。其它则showHint()显示提示词汇。
    • 提示词汇盒
      • click：显示提示词汇
      • mouseover：
      • mouseout ：
  2. showHint()函数获取输入框内的值，作为参数传递给后端。
  3. showHintText()函数在用户按下回车键后执行，在输入框中显示提示词汇

node.js源代码负责对数据库进行查询，然后用 HTML 表格返回结果

  • server：Http服务器模块，url模块获取url参数并传递给路由。

任务5：界面拖拽交互

  1. 拖放某元素时，依次触发时间入下：dragstart drag dragend

  2. 元素被拖放到有效目标位置时，依次触发事件：dragenter dragover dragleave/drop

  3. 判断拖放位置

    • 页面坐标：pageX,pageY ( 页面坐标 = 客户端信息 + 滚动信息 )
    • 客户区坐标位置：clientX,clientY
    • 滚动信息：scrollLeft,scrollRight
    • 对块级元素来说，offsetTop、offsetLeft、offsetWidth 、offsetHeight 描述了元素相对于 offsetParent 的边界框。
